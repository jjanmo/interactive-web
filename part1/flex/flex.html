📘 Container 속성
✅ flex-direction : 주축을 결정하는 속성, 주축이란 아이템들이 나열되는 방향을 말한다.
✅ justify-content : 주축에서 아이템들을 배열하는 방법에 대한 속성이다.
좀 더 자세히 말하면, 아이템 주위에 여백을 어떻게 분배할 것인지에 대한 것을 나타낸다 직접 조작하다보면 무슨 느낌인지 몸으로 인지하는게 더 좋은 방법이다
✅ align-items : 교차축에서의 아이템들의 배열하는 방벙에 대한 속성이다.
📘 Item 속성
✅ flex-grow : 아이템의 너비,아이템의 width가 아니라 컨텐츠 폭을 제외한 여백의 비율을 설정해준다
각각의 아이템마다 다른 값을 넣어주면 각각의 아이템이 서로 다른 너비를 갖게된다
✅ flex-shrink : 아이템의 찌그러지는, 축소시키는 비율
일반적으로 grow와 shrink를 다르게 사용하는 경우는 드물고 같은 값으로 설정해준다.
✅ flex-basis : 원래 컨텐츠가 차지하는 영역을 어떻게 해줄지를 설정해주는 속성으로, 기본값 auto 이다
0으로 설정할 경우 : 모든 컨텐츠의 영역 비율이 동일하게 바뀐다.
✅ 일반적으로 우리가 의도하는 바 => shorthhand : flex : 1 => flex-grow : 1/ flex-shrink : 1 / flex-basis : 0
